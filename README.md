📌 Overview:
This is my first data analysis project using Python.
The goal is to explore the Titanic dataset and understand what factors affected passenger survival.
Dataset used from Kaggle.

📊 Dataset Information:
The dataset contains information about Titanic passengers such as:
1.Passenger class (Pclass)
2.Gender (Sex)
3.Age
4.Fare
5.Survival status (Survived)

🛠️ Tools Used:
1.Python 🐍
2.Pandas 📊
3.Matplotlib 📈
4.Jupyter Notebook / Kaggle Notebook

🔍 Analysis Performed:
1.Checked dataset structure and missing values
2.Calculated overall survival rate
3.Compared survival by gender
4.Compared survival by passenger class
5.Visualized survival rates using bar charts

📈 Key Insights:
1.Females had a higher survival rate than males
2.Passengers in 1st class had better survival chances
3.Social class and gender played a major role in survival

📌 Conclusion
This project helped me understand basic data analysis using Python.
It shows how data can reveal patterns and insights about real-world events.
